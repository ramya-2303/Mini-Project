# PLY Project – Simple Syntax Parser

This mini project is built using **Python Lex-Yacc (PLY)** to perform **lexical analysis** and **syntax parsing** for a small subset of C/C++-like language constructs.

## Files
- `lexer.py` – defines tokens and lexical analyzer
- `parser.py` – contains grammar rules for syntax checking
- `main.py` – takes multi-line input and parses it

## Features
The project checks the syntax of:
- variable declarations
- function definitions
- input/output statements
- expressions
- `for` and `while` loops
- `switch-case` statements

## Technologies Used
- Python
- PLY
